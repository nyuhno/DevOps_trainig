**Поднять VM внутри VPS, поднять сетевые соединения.**
- На платформе cloud.ru поднят сервер уровня free tier
- Развернута ОС Ubuntu 24.04
- Назначен сетевой интерфейс со статичным внешним IP 
- Настроено правило для доступа по SSH - доступ из сети 0.0.0.0/0 и открытый порт 22:22
- Через netplan настроена сетевая топология внутри машины - [yaml](https://github.com/nyuhno/DevOps-trainig/blob/main/50-cloud-init.yaml) / после sudo netplan apply
- На клиентской стороне сгенерирован ssh ключ, команда ssh-keygen -t rsa -b 4096 / ключ был загружен на серверную сторону
- Подключаемся ssh user@host с внешней машины - соединение установлено, доступ внутрь машины есть
- Проверяем подключение с клиентской стороны через ssh - все работает
- Обновляем сервер sudo apt update / sudo apt upgrade / sudo apt autoremove
- Связка VPS+VM запущена, сеть настроена, VM доступна из вне по SSH.
