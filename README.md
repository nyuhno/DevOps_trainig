# DevOps_trainig
Цель получение оффера на junior DevOps или приглашение на стажировку посредством создания демо-пайплайна
План:
1. Поднять VM внутри VPS, поднять сетевые соединения.
2. Развернуть Docker и Docker Composer, развернуть внутри приложение, бд(если влезет в VPS), настроить сетевые соединения и тд.
3. Развернуть Nginx как reverse proxy.
4. Сделать CI/CD скорее всего через GitHub Actions, для авто-деплоя по push.
5. Развернуть дашборд с метриками (Prometheus + Grafana) для мониторинга состояния сервера.
