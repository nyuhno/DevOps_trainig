## DevOps_trainig
- Цель получение оффера на junior DevOps или приглашение на стажировку посредством создания демо-пайплайна.
- **План:**
- Поднять VM внутри VPS, поднять сетевые соединения. [ссылка](https://github.com/nyuhno/DevOps-trainig/blob/main/docks/VPS%2BVM.md)
- Развернуть Docker и Docker Composer, развернуть внутри приложение, бд(если влезет в VPS), настроить сетевые соединения и тд.[ссылка](https://github.com/nyuhno/DevOps-trainig/blob/main/docks/Docker%2BDockerComposer.md)
- Сделать CI/CD скорее всего через GitHub Actions, для авто-деплоя по push.
- Развернуть дашборд с метриками (Prometheus + Grafana) для мониторинга состояния сервера.
